## Overview
LoneForge was created to enable solo play without juggling dozens of scattered tables, sheets, documents, and tools.  
Everything needed for a complete adventure — character, monsters, loot, oracles, travel, downtime, notes, spells, NPCs, and more — is gathered in a single, easy‑to‑use, open‑source web tool.

---

## Key features

**Character Manager**
- Character sheet; equipment; treasures; abilities; animal companions; conditions; quick notes

**Items & Loot Generator**
- Procedural item generator; magic‑item‑linked quests; integrated loot generator; fully rewritten JSON database to avoid copyright issues

**Bestiary**
- Original monsters; stats, traits, behaviors; random encounter generator

**NPC & Factions**
- Quick NPC creation; faction generator; procedural tables for motivations, traits, goals

**Spell Compendium**
- All spells rewritten in original form; no copyrighted content; fast search and filters

**Dice Roller**
- Quick rolls; complex roll macros; result log

**Notes**
- Integrated note blocks; organization by adventure or session

**Downtime Manager**
- Crafting; training; research; between‑adventure activities

**Adventure Tools**
- Oracle; settlement generator; wilderness travel; combat encounter generator; random events

**Lore Generator**
- Procedural lore generator for fast worldbuilding
